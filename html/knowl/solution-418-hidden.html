We use inclusion and exclusion. We let \(A\) be the set of all seating arrangements. We let \(A_i\) be the set of seating arrangements such that student \(i\) sits in the same seat as before. We are interested in \(\left|\overline{\bigcup_{i=1}^n A_i}\right|\text{.}\) For this purpose, for each subset \(S\) of the set \([n]\text{,}\) we need to compute \(\left|\bigcap_{i\colon i\in S} A_i\right|\text{,}\) the number of ways for the students to return so that every student represented by an \(i\) in \(S\) sits in his or her previous seat. This leaves us with \(n-|S|\) seats to be filled in a one-to-one fashion by \(m-|S|\) students. There are \((n-|S|)^{\underline{m-|S|}}\) such seating arrangements, so \(\left|\bigcap_{i\colon i\in S} A_i\right|= (n-|S|)^{\underline{m-|S|}}\text{.}\) Thus we have

\begin{equation*} \left|\overline{\bigcup_{i=1}^n A_i}\right|=\sum_{S:S\subseteq [n]} (-1)^{|S|}(n-|S|)^{\underline{m-|S|}} = \sum_{s=0}^m (-1)^s\binom{m}{s}(n-s)^{\underline{m-s}} \end{equation*}

ways for the students to return so that nobody sits in his or her previous place.
