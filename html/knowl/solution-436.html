Solution256.b.1

The set of all permutations of \(\{1,2,3\}\) is a permutation group. Let \(\sigma\) be given by \(\sigma(1) = 2\text{,}\) \(\sigma(2) = 3\text{,}\) and \(\sigma(3) = 1\text{.}\) Let \(\varphi\) be given by \(\varphi(1) = 1\text{,}\) \(\varphi(2) = 3\text{,}\) and \(\varphi(3) = 2\text{.}\) Then \(\sigma \circ \varphi(1) = 2\text{,}\) \(\sigma\circ\varphi(2) = 1\text{,}\) and \(\sigma\circ\varphi(3) = 3\text{.}\) But \(\sigma^{-1} \circ \varphi^{-1}(2) = \sigma^{-1}(3) = 2\text{.}\) thus \((\sigma\circ\varphi)^{-1}\ne \sigma^{-1}\varphi^{-1}\text{.}\)
