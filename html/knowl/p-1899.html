Suppose the function \(f\) from \(S\) to \(T\) has an inverse \(g\text{.}\) If \(f(x) = f(y)\text{,}\) then \(g(f(x)) = g(f(y))\) and so \(x = y\text{.}\) Therefore \(f\) is one-to-one. If \(y\) is in \(T\text{,}\) then \(f(g(y)) = y\text{,}\) and so \(g(y)\) is an element \(x\) of \(S\) such that \(f(x) = y\text{.}\) Therefore \(f\) is a bijection.
