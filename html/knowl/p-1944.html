Each element is in a set \(S_i\) with itself, so the relation is reflexive. If \(x\) and \(y\) are in a given one of those sets \(S_i\text{,}\) then \(y\) and \(x\) are in that same set \(S_i\text{.}\) If \(x\) and \(y\) are in the same set \(S_i\text{,}\) and if \(y\) and \(z\) are in the same set \(S_j\text{,}\) then \(S_i\) must equal \(S_j\) because \(y\) is in one and only one block of the partition. Therefore, \(x\) and \(z\) must be in the same set \(S_i\text{.}\) Thus the relation is an equivalence relation. If \(x\in S_i\text{,}\) then by definition \(S_i\) consists of all elements related to \(x\text{,}\) so it is the equivalence class containing \(x\text{.}\)
