If \(f\) is a function from \(S\) to \(T\text{,}\) if \(g\) is a function from \(T\) to \(S\text{,}\) and if \(f(g(x))= x\) for each \(x\) in \(T\) and \(g(f(x)) = x\) for each \(x\) in \(S\text{,}\) then we say that \(g\) is an inverse of \(f\) (and \(f\) is an inverse of \(g\)).
