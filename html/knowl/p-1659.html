Every element \(y\) of a multiorbit has the same multiplicity. This is because if \(\sigma(x) = y\text{,}\) then the permutations that send \(x\) to \(y\) are the permutations \(\sigma\tau\) where \(\tau\) fixes \(x\text{.}\) Thus the size of the multiorbit is \(|\Fix(x)|\cdot |Gx|\text{,}\) so the size of the orbit divides the size of the multiorbit, which is the size of \(G\text{.}\) In particular, the size of an orbit is \(|G|/|\Fix(x)|\) for any \(x\) in the orbit.
