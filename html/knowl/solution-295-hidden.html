The vertices of degree 1 are the vertices that do not appear in the Prüfer code for the tree. So we first choose four vertices out of \(n\) in \(\binom{n}{4}\) ways to be our vertices of degree 1, and the Prüfer code may be thought of as a function from the \(n-2\) places of the code onto the \(n-4\) remaining vertices, so there are \(S(n-2,n-4)(n-4)!\) Prüfer codes for each choice of the vertices of degree 1. Thus, using Problem 3.4.6, we have that the number of labeled trees is

\begin{equation*} \binom{n}{4}\frac{(n-2)(n-4)(3n-11)(n-4)!}{24} = \frac{n!(n-2)(n-3)(n-4)(3n-11)}{576}\text{.} \end{equation*}
