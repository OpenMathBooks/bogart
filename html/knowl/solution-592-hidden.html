If we adjoin \(n\) to the subsets not containing \(n\) we get the subsets containing \(n\text{.}\) This is a bijection because if we start with two different sets, adjoining \(n\) to them can't make them the same, and every subset \(S\) containing \(n\) must arise in this way from the set \(S-\{n\}\) not containing \(n\text{.}\)
