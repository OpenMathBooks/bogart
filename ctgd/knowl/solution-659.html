Solution421.b.1

To specify a simple graph on a vertex set \(V\text{,}\) we have to specify its set of edges. The possible sets of edges thus correspond bijectively to sets of two-element subsets of \(V\text{.}\) But if \(V\) has size \(i\) the set of all two element subsets of \(V\) has \(\binom{i}{2}\) elements. Thus the number of sets of two element subsets of \(V\) is \(2^{\binom{i}{2}}\text{.}\) Therefore \(a_i = 2^{\binom{i}{2}}\text{.}\)
