\(\binom{m}{n} \lt 2^{\binom{n}{2}-1}\) is the same as \(\frac{m^{\underline{n}}}{n!}\lt 2^{\binom{n}{2}-1}\text{.}\) And since \(m^{\underline{n}}\lt m^n\text{,}\) the inequality \(\frac{m^{\underline{n}}}{n!}\lt 2^{\binom{n}{2}-1}\) holds if the inequality \(\frac{m^n}{n!}\le2^{\binom{n}{2}-1}\) holds. And this last inequality holds if \(m\le\root n \of {n!2^{\binom{n}{2}-1}}\) holds. Thus \(R(n,n)>m\) for any \(m\) such that \(m\le\root n \of {n!2^{\binom{n}{2}-1}}\text{,}\) which implies that \(R(n,n)> \root n \of {n!2^{\binom{n}{2}-1}}\text{.}\)
