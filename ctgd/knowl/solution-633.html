Solution397.1

Let \(f(x)\text{,}\) \(g(x)\text{,}\) and \(h(x)\) be, respectively, the EGFs for the species \(\F\text{,}\) \(\mathcal{G}\text{,}\) and \(\F\cdot\mathcal{G}\text{.}\) Suppose \(f(x) = \sum_{i=0}^\infty a_i \frac{x^i}{i!}\) and \(g(x) = \sum_{j=0}^\infty b_j\frac{x^j}{j!}\text{.}\) Then the coefficient of \(x^n\) in \(f(x)g(x)\) is \(\sum_{k=0}^n \frac{a_kb_{n-k}}{i!(n-k)!}\text{,}\) so the coefficient of \(\frac{x^n}{n!}\) is

\begin{equation*} n!\sum_{k=0}^n \frac{a_kb_{n-k}}{i!(n-k)!} = \sum_{k=0}^n \binom{n}{k} a_kb_{n-k}. \end{equation*}

But \(\binom{n}{k}a_kb_{n-k}\) is the number of ways to partition a set of size \(n\) into a first set of size \(k\) and a second set of size \(n-k\) and to choose a structure for each of them. That is, it is the number of pair structures on an ordered pair of sets of size \(k\) and \(n-k\text{.}\) Thus the coefficient of \(\frac{x^n}{n!}\) in \(h(x)\) is the number of \(\F\cdot\mathcal{G}\) pair structures on a subset of \(X\) of size \(n\text{.}\) This proves that \(h(x) = f(x)g(x)\text{.}\)
