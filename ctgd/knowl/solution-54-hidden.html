Suppose \(S\) and \(T\) are subsets of \([n]\text{.}\) If \(i\in S\) but \(i\not\in T\text{,}\) then \(\chi_S(i)=1\) but \(\chi_T(i)=0\text{.}\) Thus if \(S\not= T\text{,}\) the \(\chi_S\not=\chi_T\text{.}\) Therefore \(f\) is one-to-one. Given a function \(g\) from \([n]\) to \(\{0,1\}\text{,}\) let \(S=\{i|g(i)=1\}\text{.}\) Then by definition, \(g=\chi_S=f(S)\text{.}\) Therefore \(f\) is onto, so it is a bijection.
