Problem201

Recall that a partition of an integer \(k\) is a multiset of numbers that adds to \(k\text{.}\) In Problem 200 we found the generating function for the number of partitions of an integer into parts of size 1, 5, 10, and 25. When working with generating functions for partitions, it is becoming standard to use \(q\) rather than \(x\) as the variable in the generating function. Write your answers in this notation. 5 

(a)

Give the generating function for the number partitions of an integer into parts of size one through ten.
