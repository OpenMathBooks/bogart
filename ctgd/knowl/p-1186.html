The generating function for partitions that arise in the second case is \(\qchoose{m+n-1}{m-1}=\qchoose{m+n-1}{n}\text{.}\) In the first case, after we delete the largest part, the Young diagram sits in a rectangle of width \(m\) and depth \(n-1\text{.}\) The generating function for partitions that arise in the first case after we have deleted a part of size \(m\) is \(\qchoose{m+n-1}{n-1}\text{.}\) Thus the generating function for partitions that arise in the first case (before we delete the part of size \(m\)) is \(q^m\qchoose{m+n-1}{n-1}\text{.}\) Thus by the sum principle the generating function for all partitions that fit into a rectangle of width \(m\) and depth \(n\) is given by

\begin{equation*} \qchoose{m+n}{n}= q^m\qchoose{m+n-1}{n-1}+\qchoose{m+n-1}{n}. \end{equation*} in-context
