First give each recipient \(m\) objects. This leaves \(k-mn\) identical objects to be distributed among \(n\) recipients, so we may do this in the number of ways to choose a (\(k-mn\))-element multiset from \(n\) things. This is \(\binom{n+k-mn+1}{k-mn}\text{,}\) or \(\binom{k-(m-1)n+1}{n+1}\text{.}\)
