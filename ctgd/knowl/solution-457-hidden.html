Only one element can be repeated, and that element must be \(\sigma^{n+1}(i)\text{.}\) Further, we have that \(\sigma^{n+1}(i)\) must be \(i\text{,}\) because if we had

\begin{equation*} \sigma^j(i) = \sigma^{n+1}(i) \end{equation*}

and \(0 \lt j \lt n+1\text{,}\) applying \(\sigma^{-j}\) to both sides of the equation would give us an earlier repeat.
