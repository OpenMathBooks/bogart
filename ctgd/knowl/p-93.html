The number of functions from \([3]\) to \([12]\) with \(f(3) =1\) is the number of functions from \([2]\) to \([12]\text{,}\) namely 144. The same is true for the number of functions with \(f(3)=2\) and \(f(3)=3\text{,}\) and the size of the set \(S_i\text{.}\) The union of the sets \(S_i\) is the set of all functions from \([3]\) to \([12]\text{,}\) and since there are twelve sets \(S_i\text{,}\) this union has size \(12\cdot144= 1728\text{.}\)
