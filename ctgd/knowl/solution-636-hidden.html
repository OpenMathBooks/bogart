Since the sets of a \(k\)-set structure are nonepmty and disjoint, the \(k\)-element set of sets can be arranged as a \(k\)-tuple in \(k!\) ways. Thus the number of \(k\)-set structures on a given set is \(\frac{1}{k!}\) times the number of \(k\)-tuple structures on that set. Therefore the EGF for \(k\)-set structures is \(\frac{1}{k!}\) times the EGF for \(k\)-tuple structures. By CorollaryÂ C.4.2, the EGF for \(k\)-set structures is thus \(\frac{f(x)^k}{k!}\text{.}\)
