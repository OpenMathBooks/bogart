There are \(n!\) lists, and each is in an equivalence class of size \(k!(n-k)!\text{,}\) so the number of equivalence classes is \(\frac{n!}{k!(n-k)!}\) by Problem 356. This is a way of computing the number of \(k\)-element subsets that shows why the final answer we got in Problem 39 is symmetric in \(k\) and \(n-k\text{.}\)
