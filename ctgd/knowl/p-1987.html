We know that the number of subsets of a \((k-1)\)-element set is \(2^{k-1}\text{.}\) The number of subsets of \(\{1,2,\ldots,k\}\) that do not contain \(k\) is the number of subsets of the \(k-1\)-element set \(\{1,2,\ldots, k-1\}\text{,}\) so we know this number is \(2^{k-1}\text{.}\) We know that the number of subsets that do contain \(k\) equals the number that don't, so the number that do contain \(k\) is also \(2^{k-1}\text{.}\) The sum principle tells us that the number os subsets of \(\{1,2,\ldots, k\}\) is \(2^{k-1}+2^{k-1}=2^k\text{.}\) We can conclude that the statement ``for every nonnegative integer \(n\text{,}\) an \(n\)-element set has exactly \(2^n\) subsets" is true.
