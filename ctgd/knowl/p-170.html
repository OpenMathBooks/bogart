There is yet another bijection that lets us prove that a set of size \(n\) has \(2^n\) subsets. Namely, for each subset \(S\) of \([n]=\{1,2,\ldots, n\}\text{,}\) define a function (traditionally denoted by \(\chi_S\)) as follows. 1 

\begin{equation*} \chi_S(i) = \begin{cases}1 \amp \text{ if } i\in S \\ 0 \amp \text{ if } i\not\in S \end{cases} \end{equation*} in-context
