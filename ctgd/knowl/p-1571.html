The element \(k\) is either in a cycle by itself or it isn't. The number of permutations in which it is in a cycle by itself is \(c(k-1,n-1)\text{.}\) If it is in a cycle with something else, it can come after any of the \(k-1\) elements, and each choice of which one it comes after gives a different permutation.
