We expect \((1-b)\sum_{i=0}^{n-1} db^i\) to be \(d(1-b^n)\text{.}\) If \(b\not=1\text{,}\) this gives us \(\sum_{i=0}^{n-1}db^i=d\frac{1-b^n}{1-b}.\) We can prove this by induction on \(n\text{.}\) If \(n=0\) we get 0 for \(\frac{1-b^n}{1-b}\text{,}\) and also for the sum \(\sum_{i=0}^{-1}db^i\text{,}\) since that sum has no terms. Assuming the formula holds when \(n=k-1\text{,}\) we may write

\begin{align*} \amp \sum_{i=0}^{k-1} db^i\\ =\amp \sum_{i=0}^{k-2}db^i+db^{k-1}\\ =\amp d\frac{1-b^{k-1}}{1-b}+db^{k-1}\\ =\amp \frac{d-db^{k-1} +db^{k-1}-db^k}{1-b} = d\frac{1-b^k}{1-b}. \end{align*}
