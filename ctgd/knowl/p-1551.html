Suppose the (inequivalent) cycles of \(\sigma\) are \(\gamma_1, \gamma_2,\ldots,\gamma_k\text{.}\) Then let \(B_j\) be the set of elements of \(\gamma_j\text{.}\) Since each element \(i\) is in the cycle \((i\ \sigma(i)\ \ldots)\text{,}\) every element is in a set \(B_j\text{.}\) Since no two (inequivalent) cycles have an element in common, every element of \(S\) is in exactly one set \(B_j\text{.}\) Then the sets \(B_j\) are the blocks of a partition, and it is the only partition each of whose blocks is the set of elements of a cycle of \(\sigma\text{.}\) This completes the proof.
