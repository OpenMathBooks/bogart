As you see in the table, we define \(S(0,0)=1\text{,}\) and \(S(0,n)\) or \(S(k,0)\) to be 0 otherwise. This makes sense because for \(n>0\) there is no partition of an empty set into \(n\) parts, and for \(k>0\) there is no partition of a \(k\)-element set into no parts, but saying there is one partition of the empty set into no parts allows us to use our recurrence to compute \(S(1,1)\text{.}\) This makes our recurrence valid for all nonnegative values of \(k\) and \(n\text{.}\)
