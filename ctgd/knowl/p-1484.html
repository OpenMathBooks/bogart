Yes, because the permutations \(\sigma\text{,}\) \(\sigma^2\text{,}\) …, \(\sigma^n\text{,}\) … must eventually start repeating. But if \(\sigma^i=\sigma^j\text{,}\) and \(i\lt j\text{,}\) then \(\sigma^{j-i} =1\text{,}\) so \(\sigma^{j-i-1}\) is the inverse of \(\sigma\text{.}\) This means that every member of the set of permutations has an inverse in that set. But then since \(\sigma\circ\sigma^{-1} = \iota\text{,}\) we have that the identity is in the set as well. Thus it is a permutation group.
