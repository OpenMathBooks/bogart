We guess that \(1+3+5+\cdots+2n-1=n^2\text{.}\) Clearly this is true when \(n\) is 1, 2, or 3. Suppose there is an \(n\) for which this formula is not true, and let \(k\) be the smallest such \(n\text{.}\) Then \(1+3+5+\cdots+2(k-1)-1 = (k-1)^2\text{.}\) Simplifying, \(1+3+5+\cdots+2k-3 = (k-1)^2\text{.}\) Now suppose we add \(2k-1\) to both sides of this equation. Then we get

\begin{equation*} 1+3+5+\cdots+2k-3 +2k-1 = (k-1)^2+2k-1 = k^2-2k+1+2k-1=k^2. \end{equation*}

But this is a contradiction, because we assumed that \(k\) was the smallest value of \(n\) for which the sum on the left is not \(n^2\text{.}\) Therefore the assumption that there is an \(n\) for which \(1+3+5+\cdots+2n-1 \not=n^2\) must be false, so the equation \(1+3+5+\cdots+2n-1=n^2\) must be true for all positive integers \(n\text{.}\)
