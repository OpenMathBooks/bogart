We note that \(0^2=0\text{,}\) while \(2^0=1\text{,}\) that \(1^2=1\text{,}\) while \(2^1=2\text{,}\) that \(2^2=4\text{,}\) while \(2^2=4\text{,}\) that \(3^2=9\) while \(2^3=8\text{,}\) that \(4^2=16\) while \(2^4=16\text{,}\) and \(5^2=25\) while \(2^5=32\text{.}\) We suspect that \(2^n>n^2\) for \(n\ge 5\text{,}\) so we try to prove this by induction. We have already shown that \(2^5>5^2\text{.}\) Now suppose that \(k>5\) and \(2^{k-1}>(k-1)^2\text{.}\) Then \(2^k=2\cdot2^{k-1}>2(k-1)^2=2k^2-4k +1\text{.}\) Now since \(k>5\text{,}\) \(k^2>5k\text{,}\) so that \(k^2-4k+1=k^2+k^2-4k+1>k^2+5k-4k+1=k^2+k+1>k^2\text{.}\) Thus for \(k>5\text{,}\) the statement \(2^{k-1}>(k-1)^2\) implies the statement \(2^k>k^2\text{.}\) Therefore, by the principle of mathematical induction, \(2^n>n^2\) for all \(n\ge 5\text{.}\)
