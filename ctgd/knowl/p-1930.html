\(a-a=0=0\cdot n\text{,}\) so \(a\equiv a\pmod{n}\text{.}\) Thus the relation is reflexive. If \(a-b=kn\) for some integer \(k\text{,}\) then \(b-a=-kn\text{,}\) and \(-k\) is an integer, so if \(a\equiv b \pmod{n}\text{,}\) then \(b\equiv a \pmod{n}\text{.}\) If \(a-b=kn\) and \(b-c= jn\text{,}\) then \(a-b+b-c=kn+jn\text{,}\) so \(a-c=(k+j)n\) and since \(k+j\) is an integer this means that \(a\equiv c\pmod{n}\text{.}\) Therefore the relation of congruence mod \(n\) is an equivalence relation.
