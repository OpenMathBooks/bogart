Since each one-element set has five structures on it and each set of any other size has no structures on it, \(\F\) is a species. The EGF for \(\F\) is \(5x\text{,}\) because there are five ordered pairs using any given one-element set and none using any other set. Note that a set of ordered pairs whose values partition a set \(N\) of light poles \(N\) is exactly a function from the set \(N\) of light poles to the set of colors. Then by the exponential formula, the EGF for the number of ways to paint \(n\) light poles with four colors is \(e^{5x}\text{.}\)
